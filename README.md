# Git Command Simulator

A Streamlit-based web application that simulates basic Git operations in a terminal-like interface.

## Supported Commands
- `git init`
- `touch <filename>`
- `git status`
- `git add <file>` / `git add .`
- `git commit -m "message"`
- `git log --oneline`

## How to Run Locally
```bash
streamlit run app.py
```
